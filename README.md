# Infrastructure Generator

O **Infrastructure Generator** é uma ferramenta que automatiza a criação de estruturas de infraestrutura para projetos de software, permitindo gerar rapidamente configurações de servidores, redes, containers, cloud e ambientes de desenvolvimento para aplicações em .NET, Node.js, Python, Go.

## Tecnologia

- **Linguagem principal:** Go  
- **Por que da escolha:** Ferramenta de infraestrutura precisa de performance e concorrência para provisionamento, integração com cloud e automação de scripts.

## Arquitetura

- **Arquitetura recomendada:** Modular / Layered  
- **Por que da escolha:** Estruturas de infraestrutura precisam ser modulares e facilmente configuráveis para diferentes ambientes.
